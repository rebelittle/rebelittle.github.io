2026 NFL Draft Pool Tracker

April 2026 · Featured

Live scoring web app + odds-optimized draft slate. Built in 24 hours for the 2026 NFL Draft.

  • Static site on GitHub Pages with custom scoring engine (11 unit tests passing), client-side entry submission form, and JSON-driven data layer — zero backend, zero framework.
  • Ingested live DraftKings player-at-pick prop markets for 30 Round 1 slots, converted American odds to implied probabilities, and solved the global player-to-slot assignment via the Hungarian algorithm (scipy.optimize.linear_sum_assignment).
  • Optimized slate beat naive greedy baselines by +6.5 expected points under market-implied probabilities; real-time leaderboard auto-refreshes every 60 seconds.
  • Integrated Venmo deep-link payment flow for $5 entries; trade predictions score bonus points (+6 one team / +10 both teams).
